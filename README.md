# Euroopt — Self‑Checkout Kiosk Prototype

Touch‑friendly kiosk interface inspired by real self‑checkout flows. This is a **static prototype** with basic interactions (scan items, totals, assistant call, loyalty, payment modal, age check). No backend required.

## Run locally

1. Open `index.html` directly in a browser **or** use a simple server.
2. Recommended for kiosks: fullscreen and zoom 110–125%.

## Sample flows

- Scan items via barcode or PLU chips
- See item list, subtotal, and weight
- Trigger age verification for restricted items
- Proceed to payment modal
- Void transaction / call assistant / loyalty card
